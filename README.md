# ⚡ EmbedX

> 🚀 Fast, Clean, and Modern Embedded Development CLI

---

## 🚀 What is EmbedX?

EmbedX simplifies embedded development with:

- ⚡ One-command build & upload
- 🔌 Auto device detection
- 📦 Library management
- 📟 Built-in serial monitor
- 🧠 Smart build caching

---

## 📚 Documentation

👉 **Full documentation available here:**

## ⚡ Quick Start

```bash
embedx init my_project
cd my_project
embedx setup
embedx run
```

---

## 📦 Installation

```bash
git clone https://github.com/shreekanthkapparagaon/embedx.git
cd embedx
pip install -e .
```

---

## 🧰 Example Commands

```bash
embedx build
embedx upload --port COM3
embedx monitor
```
