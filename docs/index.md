# ⚡ EmbedX

<div align="center">

### 🚀 Fast, Clean, and Modern Embedded Development CLI

Build, upload, and manage embedded projects effortlessly.

</div>

---

## ⚡ Quick Start

```bash
embedx init my_project
cd my_project
embedx setup
embedx run
```

---

## 🚀 Why EmbedX?

<div class="grid cards" markdown>

- ⚡ **Fast Workflow**  
  One command to build, upload, and monitor.

- 🔌 **Auto Detection**  
  Automatically detects connected devices.

- 📦 **Smart Libraries**  
  Local + online registry with lockfile support.

- 🧠 **Efficient Builds**  
  Skips rebuilds when nothing changes.

</div>
