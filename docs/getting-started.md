# 🚀 Getting Started

## Installation

```bash
git clone https://github.com/shreekanthkapparagaon/embedx.git
cd embedx
pip install -e .
```

---

## Create Project

```bash
embedx init my_project
cd my_project
```

---

## Setup Environment

```bash
embedx setup
```

---

## Run Project

```bash
embedx run
```

---

## First Build Flow

1. Create project  
2. Setup board  
3. Build firmware  
4. Upload to device  
5. Monitor output  

---

!!! tip
    Use `embedx run` for complete workflow.
