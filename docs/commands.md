# 🧰 Commands

## 🚀 Project

| Command | Description |
|--------|------------|
| embedx init | Create new project |
| embedx build | Compile project |
| embedx run | Build + upload + monitor |
| embedx clean | Clean build files |

---

## 🔌 Device

| Command | Description |
|--------|------------|
| embedx device list | List devices |
| embedx ports | Show ports |
| embedx monitor | Serial monitor |

---

## 📦 Libraries

| Command | Description |
|--------|------------|
| embedx lib install <name> | Install library |
| embedx lib uninstall <name> | Remove library |
| embedx lib search <query> | Search libraries |

---

## ⚙️ Setup

| Command | Description |
|--------|------------|
| embedx setup | Install environment |
| embedx doctor | Diagnose issues |
